# Interactive Dashboard for Evaluating Abu Dhabi's Strategic Initiatives (2026)

## Overview
This project provides an interactive dashboard for analyzing the dataset "Evaluation of Joint Initiatives Aligned with Abu Dhabi Government's Strategic Objectives (2026)." The goal is to make the data more accessible and visually engaging for policymakers, researchers, and the general public.

## Features
- Interactive dropdown menu to filter initiatives.
- Bar charts displaying planned vs. actual budgets.
- Easy-to-use interface for users with varying technical expertise.

## Requirements
- Python 3.7+
- Dash
- pandas
- plotly
- OpenPyXL

## Installation
1. Clone the repository:
    
    git clone https://github.com/your-repo/abu-dhabi-dashboard.git
    cd abu-dhabi-dashboard
    

2. Install dependencies:
    
    pip install -r requirements.txt
    

3. Run the application:
    
    python app.py
    

4. Open your browser and navigate to `http://127.0.0.1:8050/`.

## Usage
1. Select an initiative from the dropdown menu to view specific data.
2. Analyze the visualized data for planned vs. actual budgets.
